Desigualdad CSCH#

Enn 1970 Clauser, Horne, Shimony y Holt [12] propusieron una figura de m√©rito f√°cilmente accesible para verificar las desigualdades de Bell.   1. Alice y Bob seleccionan cada uno una orientaci√≥n (de las dos posibles de cada uno), p. ej. \(\hat{\bf n}_A\) y \(\hat{\bf n}'_B\), para sus detectores. Hay cuatro parejas posibles dependiendo de que selecciones cada uno

(443)#\[\begin{equation} C(\hat{\bf n}_A, \hat{\bf n}'_B) = \frac{1}{N}\sum_{i=1}^N a_i b'_i ~\in~ [-1,1] \end{equation}\]

  1. Repiten todo el proceso anterior para las cuatro posibles orientaciones elegidas de forma aleatoria. Con las \(4N\) mediciones construyen la cantidad

(444)#\[\begin{equation} R = | C(\hat{\bf n}_A, \hat{\bf n}_B) + C(\hat{\bf n}_A, \hat{\bf n}'_B) + C(\hat{\bf n}'_A, \hat{\bf n}_B)- C(\hat{\bf n}'_A, \hat{\bf n}'_B)| \end{equation} \]

En un mundo cl√°sico, supondr√≠amos que los valores \(a_i,b_i\) proceden de valores predefinidos para cada sistema individual, sobre el que efectuamos simplemente un promedio estad√≠stico de muchos sistemas. Entonces podemos probar la desigualdad de CSCH:

Teorema:

La desigualdad de CSCH afirma que

(445)#\[\begin{equation} R \leq 2 \end{equation} \]

(446)#\[\begin{equation} a_i (b_i + b'_i) + a'_i (b_i - b'_i) = \pm 2 \end{equation}\]

porque si \(b_i + b'_i=\pm 2\) entonces \(b_i - b'_i=0\) y viceversa. Ahora podemos demostrar la desigualdad

\[\begin{eqnarray*} R &=& \lim_{N\to \infty}|\frac{1}{N} \sum_{i=1}^N \left( a_i b_i + a_i b'_i + a'_i b_i - a'_i b'_i\right) | \nonumber\\ &=& \lim_{N\to \infty}\frac{1}{N}| \sum_{i=1}^N \left( a_i (b_i + b'_i) + a'_i (b_i - b'_i)\right)| \nonumber\\ &\leq & \lim_{N\to \infty} \frac{1}{N} \sum_{i=1}^N |\left( a_i (b_i + b'_i) + a'_i (b_i - b'_i)\right)| \ \nonumber\\ &= & \lim_{N\to \infty} \frac{1}{N} \sum_{i=1}^N 2 \ \nonumber\\ &=& 2 \end{eqnarray*}\]
