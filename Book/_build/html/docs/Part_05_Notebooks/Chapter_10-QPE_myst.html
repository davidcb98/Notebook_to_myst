33.1. Ejemplo: Un solo autoestado en la ancilla#

Vamos a estudiar esta distribuci贸n analizando el operador unitario de fase

\[\begin{split} U = \begin{bmatrix} e^{2\pi i \phi_0} & 0 \\ 0 & e^{2\pi i\phi_1}\end{bmatrix} \end{split}\]

que tiene por autoestados \(| 0 \rangle \) y \(| 1\rangle \) con autovalores \(e^{2\pi\phi_0}\) y \(e^{2\pi \phi_1}\) respectivamente.

Para programar este operador notamos que

\[\begin{split} U = \begin{bmatrix} e^{2\pi \phi_0} & 0 \\ 0 & e^{-2\pi i\phi_0}\end{bmatrix} \begin{bmatrix} 1 & 0 \\ 0 & e^{2\pi i(\phi_0+\phi_1)}\end{bmatrix} = R_z(\theta = -4\pi \phi_0) \cdot P(\theta = 2\pi (\phi_1+\phi_0)) \end{split}\]

El circuito requiere la versi贸n controlada \(CU^k\) de potencias de este operador

def c_U_gate(phi0, phi1, power):
    U = QuantumCircuit(1)        
    for _ in range(power):
        U.rz(-phi0*4*np.pi, 0)
        U.p((phi1+phi0)*2*np.pi, 0)
    U = U.to_gate()
    U.name = "c_U(%f ,%f)^%i" % (np.round(phi0,5), np.round(phi1,5),power)
    U = U.control()
    return U
