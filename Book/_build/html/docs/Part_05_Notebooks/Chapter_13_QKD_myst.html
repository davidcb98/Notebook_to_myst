El BB84 de QKD implica repetir este proceso suficientes veces como para que un esp칤a tenga una probabilidad insignificante de salirse con la suya en esta interceptaci칩n. El protocolo es el siguiente:

  • Paso 1: Alice elige dos cadenas aleatorias de 4N-bits.

    \[\begin{split} \begin{array}{ll} 1001101\dots \\ 0011101\dots \end{array} \end{split}\]

  • Paso 2: Alice usa estas cadenas para codificar los estado: la primera le dir치 si tiene que enviar un \(0\) o un \(1\) y la segunda le dir치s si ese valor tiene que ir codificado en la base \(Z\) (si es \(0\) en la segunda) o en la base \(X\) (si es \(1\) en la segunda). Es decir:

    \[\begin{split} \left. \begin{array}{ll} \text{Valores}: & 1001101\dots \\ \text{Base}: & 0011101\dots \end{array} \right\} \Rightarrow |1\rangle |0 \rangle | +\rangle |- \rangle |- \rangle | 0\rangle |- \rangle \dots \end{split}\]

    Este mensaje es enviado a Bob.

  • Paso 3: Bob genera una cadena aleatoria de 4N-bits. Esta cadena le dir치 si tiene medir en \(Z\) o en \(X\) los estados que Alice le envi칩. Una vez medido, Bob mantiene este resultado privado.

  • Paso 4: Bob y Alice comparten p칰blicamente qu칠 base han utilizado para cada q칰bit. Si Bob midi칩 un q칰bit en la misma base en la que Alice lo prepar칩, lo utilizan para formar parte de su clave secreta compartida; de lo contrario, descartan la informaci칩n de ese bit. En promedi칩, la clave en este punto tendr치 2N-bits.

  • Paso 5: Por 칰ltimo, Bob y Alice comparten una muestra aleatoria de sus claves y, si las muestras coinciden, pueden estar seguros (con un peque침o margen de error) de que su transmisi칩n se ha realizado correctamente. Esta muestra puede ser, por ejemplo, de N-bits, dejando una clave secreta de tambi칠n N-bits.

    message = []
    for i in range(n):
        qc = QuantumCircuit(1,1)
        if bases[i] == 0: # Prepare qubit in Z-basis
            if bits[i] == 0:
                pass 
            else:
                qc.x(0)
        else: # Prepare qubit in X-basis
            if bits[i] == 0:
                qc.h(0)
            else:
                qc.x(0)
                qc.h(0)
        qc.barrier()
        message.append(qc)
    return message

    measurements = []
    for q in range(n):

        if bases[q] == 1: # measuring in X-basis

        measured_bit = int(result.get_memory()[0])
        measurements.append(measured_bit)
    return measurements
