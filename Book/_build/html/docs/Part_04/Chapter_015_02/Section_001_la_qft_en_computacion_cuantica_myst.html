Nota (base computacional y base de fourier)

Independientemente de la ‚Äúetiqueta‚Äù que haya dentro del ket, estaremos tratando con elementos base computacional cuando el elemento no lleve tilde, es decir, \(\ket{x}\), \(\ket{j}\), \(\dots\). Por otro lado, reservamos las letras griegas para referirnos a estados, es decir, combinaciones lineales de elementos de la base: \( \ket{\Psi} = \sum_{j=0}^{N-1} \psi_j \ket{j}\)

Veamos como act√∫a la QFT sobre los elementos de la base computacional. Para ello, solo tenemos que tomar \(\ket{\Psi}\) como un elemento de la base en la Ec. (15.3). Es decir

(15.4)#\[\begin{equation} \ket{\Psi} = \sum_{j=0}^{N-1} \psi_j \ket{j} \xrightarrow[\psi_x = 1]{\psi_j = 0 \,\,\, \forall j \neq x} \ket{\Psi} = \ket{x} \end{equation}\]

(15.6)#\[\begin{equation} \boxed{\omega = e^{2\pi i /N}} \, , \hspace{1cm} x = 0,...,N-1 \, , \hspace{1cm} \widetilde{x} = \widetilde{0}, \dots, \widetilde{N-1} \, ,\hspace{0.7cm} \text{y} \hspace{0.7cm} N=2^n. \end{equation}\]

La Ec. (15.5) es la transformada de Fourier de un elemento de la base computaci√≥nal \(\ket{x}\). Esta nos da el elemento transformado en la base de Fourier \(\ket{\widetilde{x}}\). Como podemos ver, la transformada de Fourier es un operador unitario. Como tambi√©n sabemos, un operador est√° completamente definido si especificamos como act√∫a sobre todos los elementos de una base, como es el caso.

Nota (QFT y Walsh-Hadamard)

(15.7)#\[\begin{equation} W: \ket{x} \to \ket{\tilde x} = \frac{1}{\sqrt{N}}\sum_{y}(-1)^{xy}\ket{y} = \frac{1}{\sqrt{N}}\sum_{y} e^{i\pi xy} \ket{y} \end{equation}\]

Como vemos, esta suma en \(y\) es cero para todos los valores de \(x\) excepto para \(x = 0, N, 2N, \dots\).
